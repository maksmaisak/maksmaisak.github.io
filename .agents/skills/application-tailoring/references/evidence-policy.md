# Evidence Policy

## Candidate claims

Candidate claims must be traceable to repository-backed facts.

Allowed sources include:

- `job-search/facts/`
- approved public site files such as `index.html`, `portfolio.html`, and `metro-awakening.html`
- application source files previously created from the same evidence base

## External research

External research is allowed for:

- company background
- role context
- terminology alignment
- tailoring emphasis

External research is not allowed to create or strengthen personal claims about Maks.

## When to ask

Ask the user instead of inferring when:

- multiple repository sources support different phrasings
- a metric is implied but not explicitly present
- the wording could materially change perceived scope or seniority
- a claim sounds stronger than the evidence actually supports
